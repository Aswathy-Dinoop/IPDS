# നെറ്റ്‌വർക്ക് ഇൻട്രൂഷൻ ഡിറ്റക്ഷൻ സിസ്റ്റം (NIDS)
## യൂണിവേഴ്സിറ്റി പ്രോജക്റ്റ് വിവരണം (മലയാളം)

### 1. പ്രോജക്റ്റ് അവലോകനം (Overview)
**പേര്:** Network Intrusion Detection and Prevention System using CNN.

**വിശദീകരണം:**
ഇതൊരു **ആർട്ടിഫിഷ്യൽ ഇന്റലിജൻസ് (AI)** അധിഷ്ഠിത സുരക്ഷാ സംവിധാനമാണ്. ഇന്റർനെറ്റിലൂടെയോ നെറ്റ്‌വർക്കിലൂടെയോ വരുന്ന ഹാക്കിംഗ് ശ്രമങ്ങളെയും ആക്രമണങ്ങളെയും (Cyber Attacks) തത്സമയം കണ്ടുപിടിക്കാനും തടയാനും ഈ പ്രോജക്റ്റ് സഹായിക്കുന്നു. സാധാരണ ആന്റിവൈറസുകൾക്ക് കണ്ടുപിടിക്കാൻ കഴിയാത്ത പുതിയ തരം ആക്രമണങ്ങളെപ്പോലും (Zero-day attacks) ഇതിലെ **Deep Learning (CNN)** സാങ്കേതികവിദ്യ ഉപയോഗിച്ച് തിരിച്ചറിയാൻ സാധിക്കും.

ഇതിനൊരു **Web Dashboard** ഉണ്ട്. അതിലൂടെ നമുക്ക് മൊത്തം ആക്രമണങ്ങളുടെ കണക്കുകളും വിവരങ്ങളും കാണാൻ സാധിക്കും.

### 2. പ്രോജക്റ്റ് എങ്ങനെ റൺ ചെയ്യാം? (How to Run)

**ആവശ്യമായ കാര്യങ്ങൾ (Prerequisites):**
- Python 3.8 അല്ലെങ്കിൽ അതിനു മുകളിലുള്ള വേർഷൻ.
- Administrator (Admin) അനുമതി ആവശ്യമാണ്.

**Step-by-Step നിർദ്ദേശങ്ങൾ:**

1.  **ആവശ്യമായ ഫയലുകൾ ഇൻസ്റ്റാൾ ചെയ്യുക:**
    പ്രോജക്റ്റ് ഫോൾഡറിൽ ടെർമിനൽ തുറന്ന് താഴെ പറയുന്ന കമാൻഡ് ടൈപ്പ് ചെയ്യുക:
    ```bash
    pip install -r requirements.txt
    ```

2.  **AI മോഡൽ പരിശീലിപ്പിക്കുക (Training):**
    ആക്രമണങ്ങൾ തിരിച്ചറിയാൻ സിസ്റ്റത്തെ ആദ്യം പഠിപ്പിക്കേണ്ടതുണ്ട്. അതിനായി ഈ കമാൻഡ് റൺ ചെയ്യുക:
    ```bash
    python train_model.py
    ```
    *(ഇത് `cnn_model.h5` എന്നൊരു ഫയൽ ഉണ്ടാക്കും).*

3.  **പ്രോജക്റ്റ് സ്റ്റാർട്ട് ചെയ്യുക:**
    ഡാഷ്‌ബോർഡ് ഓപ്പൺ ചെയ്യാൻ ഈ കമാൻഡ് നൽകുക:
    ```bash
    python app.py
    ```
    *ഇപ്പോൾ ബ്രൗസറിൽ `http://127.0.0.1:5000` എന്ന അഡ്രസ്സ് ഓപ്പൺ ചെയ്യുക. അവിടെ "ACTIVATE DEFENSE SYSTEM" എന്ന ബട്ടൺ ക്ലിക്ക് ചെയ്യുക.*

4.  **ആക്രമണങ്ങൾ പരീക്ഷിക്കാൻ (Testing):**
    ഇതൊരു പ്രോജക്റ്റ് ആയതുകൊണ്ട്, യഥാർത്ഥ ഹാക്കർമാർ ആക്രമിക്കുന്നത് വരെ കാത്തിരിക്കാൻ കഴിയില്ല. അതുകൊണ്ട് നമുക്ക് തന്നെ ചില ആക്രമണങ്ങൾ ഉണ്ടാക്കി പരിശോധിക്കാം. ഇതിനായി **പുതിയൊരു ടെർമിനൽ** (New Terminal) തുറന്ന് താഴെ പറയുന്ന കമാൻഡ് റൺ ചെയ്യുക:
    ```bash
    python attack_simulation.py
    ```
    *എന്നിട്ട് 1 അല്ലെങ്കിൽ 2 അമർത്തി DoS Attack അല്ലെങ്കിൽ Port Scan തിരഞ്ഞെക്കുക. ഡാഷ്‌ബോർഡിൽ അലേർട്ടുകൾ വരുന്നത് കാണാം!*

### 4. ഉപയോഗിച്ച അൽഗോരിതം: 1D CNN

ഈ പ്രോജക്റ്റിൽ നമ്മൾ ഉപയോഗിച്ചിരിക്കുന്നത് **1D Convolutional Neural Network (CNN)** ആണ്. സാധാരണയായി CNN ഇമേജുകൾ (ഫോട്ടോകൾ) തിരിച്ചറിയാനാണ് ഉപയോഗിക്കുന്നത്. എന്നാൽ ഇവിടെ നെറ്റ്‌വർക്ക് ട്രാഫിക് ഡാറ്റ വിശകലനം ചെയ്യാൻ നമ്മൾ ഇതിനെ മാറ്റം വരുത്തി ഉപയോഗിക്കുന്നു.

**ഇതെങ്ങനെ പ്രവർത്തിക്കുന്നു?**
1.  ഒരു നെറ്റ്‌വർക്ക് കണക്ഷനിലെ വിവരങ്ങളെ (Data packets) ഇത് 41 പ്രത്യേക പോയിന്റുകളായി (Features) മാറ്റുന്നു.
2.  **Conv1D Layer**: ഈ ഡാറ്റയിലെ പാറ്റേണുകൾ ഇത് പരിശോധിക്കുന്നു. ഉദാഹരണത്തിന്, ഒരേ സമയം ഒരുപാട് കണക്ഷൻ വരുന്നുണ്ടോ എന്ന് ഇത് നോക്കും.
3.  **Dense Layer**: അവസാനം, ഈ പാറ്റേണുകൾ വെച്ച് ഇത് തീരുമാനിക്കും - ഇത് "നല്ല ട്രാഫിക്" (Normal) ആണോ അതോ "ആക്രമണം" (Attack) ആണോ എന്ന്.

### 5. വളരെ പ്രധാനപ്പെട്ട ചോദ്യം: എന്തുകൊണ്ട് Random Forest-ന് പകരം CNN ഉപയോഗിച്ചു?

വൈവ (Viva) സമയത്ത് ചോദിക്കാൻ സാധ്യതയുള്ള പ്രധാന ചോദ്യമാണിത്. ഇതിനുള്ള മറുപടി:

1.  **ഓട്ടോമാറ്റിക് പഠനം (Automated Feature Extraction):**
    Random Forest ആൽഗോരിതം ഉപയോഗിക്കുമ്പോൾ, നമ്മൾ തന്നെ ഡാറ്റയിലെ ഓരോ ചെറിയ കാര്യങ്ങളും (Features) എടുത്തു പറഞ്ഞു കൊടുക്കണം. എന്നാൽ **CNN** അങ്ങനെയല്ല. അത് ഡാറ്റയിൽ നിന്ന് സ്വയം കാര്യങ്ങൾ പഠിച്ചെടുക്കും. സങ്കീർണ്ണമായ ആക്രമണങ്ങൾ കണ്ടെത്താൻ ഇത് കൂടുതൽ നല്ലതാണ്.

2.  **തുടർച്ചയായ വിവരങ്ങൾ (Sequential Data):**
    നെറ്റ്‌വർക്ക് ട്രാഫിക് എന്നത് ഒന്നിന് പുറകെ ഒന്നായി വരുന്ന വിവരങ്ങളാണ്. ഇത്തരം തുടർച്ചയായ വിവരങ്ങൾ (Sequential Data) വിശകലനം ചെയ്യാൻ Random Forest-നേക്കാൾ മികച്ചത് CNN ആണ്.

3.  **കൃത്യതയും ആധുനികതയും (Accuracy & Modernity):**
    പഴയ മെഷീൻ ലേണിംഗ് രീതികളേക്കാൾ (Decision Tree, Random Forest) കൂടുതൽ കൃത്യത Deep Learning (CNN) നൽകുന്നു. കൂടാതെ, ഇതൊരു ഫൈനൽ ഇയർ പ്രോജക്റ്റ് ആയതുകൊണ്ട്, ഏറ്റവും പുതിയ സാങ്കേതികവിദ്യ (Deep Learning) ഉപയോഗിക്കുന്നത് പ്രോജക്റ്റിന്റെ നിലവാരം ഉയർത്തുന്നു.

### 6. പ്രോജക്റ്റിന്റെ പ്രത്യേകതകൾ (Features)
*   **തത്സമയ സംരക്ഷണം (Real-time Detection):** ആക്രമണം നടക്കുന്ന സമയത്ത് തന്നെ അത് കണ്ടുപിടിക്കുന്നു.
*   **ഓട്ടോമാറ്റിക് ബ്ലോക്കിംഗ് (Auto Block):** ഹാക്കറുടെ IP Address തനിയെ ബ്ലോക്ക് ചെയ്യാനുള്ള കഴിവ്.
*   **ലളിതമായ ഡാഷ്‌ബോർഡ്:** സാങ്കേതിക അറിവില്ലാത്തവർക്കും കാര്യങ്ങൾ മനസ്സിലാക്കാം.
